# Hospital Management System

A Java console-based Hospital Management System built with OOP principles. The system allows hospital staff to manage patients, doctors, departments, appointments, and billing — with data persistence across sessions.

## Features

- **Patient Management** – Register and look up patients with medical history
- **Doctor Management** – Add doctors with specializations
- **Department Management** – Create departments and assign doctors to them
- **Appointment Scheduling** – Book appointments between patients and doctors
- **Billing System** – Generate itemized bills for appointments with predefined hospital services
- **Reports** – View reports for patients, doctors, appointments, billing, and departments
- **Data Persistence** – All data is saved and loaded automatically via binary file serialization

## OOP Concepts Used

- Inheritance (`Person` → `Patient`, `Doctor`)
- Interfaces (`Payable`)
- Encapsulation
- Enums (`Gender`, `AppointmentStatus`)
- Serialization for data persistence

## Technologies

- Java
- Eclipse IDE

## How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/your-username/hospital-management-system.git
   ```
2. Open the project in Eclipse
3. Run `MainApp.java`
